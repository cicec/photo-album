# 更新日志

## 开发测试阶段

开发测试阶段日志主要记录项目的开发进度与过程。

### 测试版 0.1.0 - 2018-07-23

#### 新增

- 「 用户验证 」基本功能，包括：登录、注册、修改信息、注销；
- 「 相册管理 」基本功能，包括：浏览、新建、删除；
- 「 图片管理 」基本功能，包括：浏览、上传、删除。

### 测试版 0.1.1 - 2018-10-17

#### 新增

- 「 信息提示 」与「 确认操作 」相关组件。

### 测试版 0.1.2 - 2018-10-31

#### 新增

- 点击图片显示「 查看详情 」页面；
- 弹出式组件与相册中图片入场与离场的过渡动画。
